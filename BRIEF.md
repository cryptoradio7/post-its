# BRIEF — Post-its
Date : 2026-03-20
Type : coding

## Description
Application de post-its memo sur le bureau Linux. Permet de creer des notes
collantes, ecrire dedans a la volee, et les retrouver au redemarrage.

## Specifications projet
- Type : Logiciel desktop
- Utilisateurs : Moi seul
- Environnement : Local (Linux Ubuntu)
- Contraintes : Aucune
- MVP : Creer un post-it, ecrire dedans, le voir sur le bureau, persister au redemarrage, supprimer
- Qualite : Prototype rapide (Dev + QA leger)
- Git : GitHub / cryptoradio7
- Chemin : ~/Bureau/APPS/post-its/
- Email : emmanuelgenesteix@yahoo.fr
- Implication : Autonome (1 gate : brief uniquement)
- Assets : Screenshot Sticky Notes Windows (references/assets/sticky_notes_ref.png)

## Reference visuelle
L'app doit ressembler aux Sticky Notes Windows : post-its colores superposables
sur le bureau, avec couleurs differentes (jaune, vert, bleu, rose).
